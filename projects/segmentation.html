#interactive_segmentation

Deep neural networks have become a mainstream approach to interactive segmentation. The major goal of our research is finding a proper balance of accuracy, network complexity, and resource consumption.

First, we developed AdaptIS (Adaptive Instance Selection) network for class-agnostic instance segmentation . Given an input image and a point, it generates a mask for the object located at this point using AdaIN layers. AdaptIS can be easily combined with standard semantic segmentation pipeline to perform panoptic segmentation. AdaptIS sets a new state-of-the-art on Cityscapes and Mapillary, and shows competitive results on a challenging COCO dataset.

Aiming for faster inference, we proposed f-BRS (feature backpropagating refinement scheme) that solves an optimization problem with respect to auxiliary variables instead of the network inputs, and requires running forward and backward pass just for a small part of a network. Experiments on GrabCut, Berkeley, DAVIS and SBD datasets set new state-of-the-art at an order of magnitude lower time per click compared to the original BRS.

However, inference-time optimization schemes are considerably more computationally expensive compared to feedforward approaches since they require performing backward passes through a network during inference. Moreover, they are difficult to deploy on mobile frameworks that usually support only forward passes. So, we proposed a simple feedforward model for click-based interactive segmentation that employs the segmentation masks from previous steps. It allows not only to segment an entirely new object, but also to start with an external mask and correct it.
